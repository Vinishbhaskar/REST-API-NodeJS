# REST-API-NodeJS
 
# REST API NodeJs
Firstly, Clone this repo using the command
```
$ git clone https://github.com/Vinishbhaskar/REST-API-NodeJS.git
```

> Required  Dependencies:
- > express
- > mongoose
- > validator

## Install Dependencies:
```
npm i express
```
```
npm i mongoose
```
```
npm i validator
```

## To run
```
npm run dev
```
